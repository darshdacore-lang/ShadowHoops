# 🏀 Shadow Hoops

A basketball game project built with Swift and Xcode.

## 🎮 About

Shadow Hoops is a basketball game created as a personal development project.  
The goal is to experiment with game mechanics, player movement, scoring systems, and UI design while improving Swift and iOS development skills.

## ✨ Features

- 🏀 Basketball gameplay mechanics
- 🎯 Shooting system
- 🕹️ Player controls
- 📱 Built for Apple platforms
- ⚡ Powered by Swift and Xcode

## 🛠️ Technologies

- **Language:** Swift
- **Engine/Framework:** Swift + Xcode
- **Platform:** Apple platforms (iOS/macOS)
